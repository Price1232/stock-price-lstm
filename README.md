# stock-price-lstm
This project uses a deep learning LSTM model to predict stock prices using historical market data.
Technologies
Python
TensorFlow
Keras
Pandas
NumPy
Matplotlib

Features
Time-series preprocessing
LSTM neural network
Stock price prediction visualization

## Model Architecture

Input: Time-series window of historical stock prices  
Model: LSTM Neural Network  
Layers: LSTM → Dense → Output  

Loss Function: Mean Squared Error  
Optimizer: Adam
